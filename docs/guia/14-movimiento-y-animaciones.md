# Movimiento y animaciones

CopyNotes tiene animaciones suaves y rápidas. No cambian nada de lo que ves
en reposo: mismos colores, tamaños y lugares. Solo cambian **cómo aparecen,
desaparecen y responden** las cosas. Todo es tranquilo y breve; nada rebota
ni se mueve solo, y nada te retrasa al escribir.

## Qué vas a notar

- **La lista lateral en el teléfono**: cuando abrís la lista de notas, entra
  deslizándose desde el costado, y el fondo oscuro aparece con un fundido
  suave. Al cerrarla, se va igual de suave.
- **En la computadora**: la lista aparece al instante, sin deslizarse, para
  que el texto y el cursor del editor nunca se muevan de lugar.
- **Carpetas**: la flechita gira al abrir o cerrar una carpeta.
- **Arrastrar para ordenar**: al tomar una nota, snippet o etiqueta, se
  levanta apenas (con una sombra suave) para que se note cuál estás moviendo.
- **Favoritos**: la estrella responde con un pequeño gesto al tocarla.

## "Reducir movimiento"

Si activás **"Reducir movimiento"** en tu equipo (macOS, iPhone o el
navegador), CopyNotes lo respeta solo: las cosas aparecen al instante, sin
deslizarse ni crecer. No perdés ninguna función; simplemente no hay
movimiento. Es una opción del sistema pensada para quien se marea o se
distrae con las animaciones.

Está en:

- **macOS**: Ajustes del sistema → Accesibilidad → Pantalla → Reducir
  movimiento.
- **iPhone/iPad**: Ajustes → Accesibilidad → Movimiento → Reducir
  movimiento.
