# Comentario del renglón

## Comentario del renglón (estilo Workflowy)

Cada renglón puede tener un **comentario**: un texto secundario, gris y más chico, pegado debajo, para aclaraciones o detalles que no querés que ensucien la línea principal. (Antes lo llamábamos "nota gris"; le cambiamos el nombre para no confundirlo con la **Nota** que ves en la lista de la izquierda.)

- **Agregar/editar:** parado en un renglón, apretá **Ctrl/Cmd+Enter**. Se abre el comentario justo abajo y escribís ahí. Puede tener varias líneas (un Enter hace una línea nueva dentro del comentario). En teléfono o tablet, usá el menú **⋯ → Agregar comentario**.
- **Salir del comentario:** apretá **Enter dos veces seguidas** y seguís escribiendo en un renglón nuevo, abajo de todo, como si nada. También podés hacer clic en otro lado, o **Escape** para volver al renglón.
- **Borrar el comentario:** con el comentario vacío, apretá **Backspace**: desaparece y volvés al renglón. Un comentario vacío que perdés el foco también se esconde solo.
- El comentario **viaja con el renglón**: se guarda, se copia (como sub-línea debajo) y se exporta en Markdown y HTML.

> **Ojo, cambió:** antes el comentario se abría con Shift+Enter. Ahora Shift+Enter hace un **salto de línea dentro del mismo renglón**, y el comentario se abre con **Ctrl/Cmd+Enter**.
